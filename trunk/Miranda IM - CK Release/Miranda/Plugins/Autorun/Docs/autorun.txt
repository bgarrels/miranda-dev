with system startup.

Usage
------
Goto: Options->Plugins->Autorun

Translation
-----------
"Autorun" can be translated with the Miranda IM language files


;; == Autorun v0.0.0.3 ==

;Name of plugin
;[Autorun]

;Options page string
;[Launch Miranda IM at system startup]

;Options page string
;[Note: At startup Miranda IM will use current profile]
