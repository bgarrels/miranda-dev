Gender plugin for Miranda IM (http://miranda-im.org)
Homepage: http://thief.miranda.im
Version: 0.0.2.1 (c) 2006-2011 Thief
Idea by nile, icons by a0x

This plugin shows gender icon in contact list. Nothing more ;)
